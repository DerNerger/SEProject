Detailed Description

This class is used to wrap and return a physical connection within a logical handle. It also registers and notifies ConnectionEventListeners of any ConnectionEvents

void com.mysql.jdbc.jdbc2.optional.JDBC4MysqlPooledConnection.addStatementEventListener ( StatementEventListener  listener)

Registers a StatementEventListener with this PooledConnection object. Components that wish to be notified when PreparedStatements created by the connection are closed or are detected to be invalid may use this method to register a StatementEventListener with this PooledConnection object.

Parameters
listeneran component which implements the StatementEventListener interface that is to be registered with this PooledConnection object
